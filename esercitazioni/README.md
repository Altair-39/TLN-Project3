## Istruzioni per l'esecuzione dei programmi 

Per le esercitazioni:

- Creare un ambiente virtuale e attivarlo

```bash

python -m venv venv

source venv/bin/activate

```

- Installare i requirements

```bash

pip install -r requirements.txt

```
